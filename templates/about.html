About This Project

This project is a simulation of a multimodal biometric system.

The purpose of this web application is to demonstrate the user interface (UI) and user experience (UX) of a modern identification system. It uses image uploads to represent face and fingerprint verification.

How It Works

Instead of using complex artificial intelligence to analyze the biometric data within the images, this prototype uses a simple verification method. When a user registers (a feature we can add later), the system stores a reference to their images. When they try to "verify," the system simply checks if the newly uploaded image file matches the stored one.

Technologies Used:

  • Backend: Flask (Python)
  • Frontend: HTML, CSS, and Vanilla JavaScript
  • Database: SQLite (for storing user/image references)
